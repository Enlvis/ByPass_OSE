<h1 align="center">OSE 🌀 Bypass</h1>

<p>
Ogólnopolska sieć edukacyjna, czyli <b>OSE</b>, to system dla szkół,
który filtruje i blokuje niektóre strony internetowe
(np. wybrane serwisy online, narzędzia webowe itp.).
</p>

<h2>Usługi OSE:</h2>
<ul>
  <li><b>OSE Internet 🛜</b> – katalogowanie stron, blokowanie przez DNS oraz dostęp do internetu dla szkół</li>
  <li><b>Certyfikat OSE 🔗</b> – aplikacja / certyfikat instalowany na komputerze szkolnym do weryfikacji urządzenia</li>
</ul>

<h2>Jak OSE blokuje strony?</h2>

<p>
OSE blokuje strony na poziomie usługi <b>"OSE Internet for School"</b>.
Działa to częściowo podobnie do VPN, choć technicznie nie jest to klasyczny VPN.
Uproszczone porównanie:
</p>

<p><b>VPN:</b><br>
Komputer → DNS → VPN → Strona internetowa
</p>

<p><b>OSE:</b><br>
Komputer → DNS → Centralny serwer OSE (sprawdzenie czy strona jest zablokowana) → Strona
</p>

<ul>
  <li>Jeśli domena nie jest zablokowana → strona otwiera się normalnie</li>
  <li>Jeśli domena jest zablokowana → użytkownik dostaje stronę z komunikatem o blokadzie</li>
</ul>
